# MSTRY MANAGEMENT

Modern Next.js website for MSTRY MANAGEMENT, built with TypeScript, Tailwind CSS, and the App Router.

## Structure

- `src/app` - App Router pages and global layout
- `src/components/layout` - Header, footer, and shell components
- `src/components/sections` - Marketing and SaaS-style page sections
- `src/components/ui` - Reusable UI primitives
- `src/config` - Brand, navigation, services, FAQs, and content data
- `public/assets` - Logo and visual assets

## Run

Install dependencies with your package manager, then run:

```bash
npm run dev
```

The current environment has Node installed but no package manager available, so dependencies were not installed here.
